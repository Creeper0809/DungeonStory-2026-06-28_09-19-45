# 몬스터 종족 컨셉

몬스터 종족은 단순 외형 분류가 아니다.

각 종족은 다음 네 가지를 동시에 결정한다.

- 어떤 고객층을 유치할 것인가
- 어떤 직원을 영입할 수 있는가
- 어떤 방어/원정 전략과 이어지는가
- 어떤 내부 사고와 운영 부담을 감수해야 하는가

즉 종족은 `경영 + 직원 운영 + 디펜스 + 오펜스`를 잇는 핵심 축이다.

## 기본 원칙

종족은 장점만 있으면 안 된다.

좋은 종족 컨셉은 항상 다음 구조를 가진다.

```text
잘 만족시키면 강한 보상
-> 단골/영입 가치 증가
-> 하지만 운영 리스크도 증가
```

종족별로 달라야 하는 값:

- 고객 가치: 돈을 많이 쓰는가, 자주 오는가
- 주요 욕구: 식사, 쇼핑, 휴식, 훈련, 마력, 도박 등
- 선호 시설: 만족도와 단골도를 올리는 시설
- 기피 시설: 만족도를 떨어뜨리는 시설/환경
- 직원 적성: 영입 후 어떤 작업에 강한가
- 방어 적성: 침입/반란 대응에서 어떤 역할을 하는가
- 원정 적성: 오펜스에 보냈을 때 어떤 장점이 있는가
- 사고 위험: 만족도가 낮을 때 어떤 내부 문제를 만드는가
- 시설 공격 컨셉 시너지: 어떤 방어 시설/속성과 어울리는가

## 종족 선택 방향

종족 로스터는 외형 인상보다 게임에서 맡는 역할이 먼저 읽혀야 한다.

캐릭터 외형 선택 폭은 고려하되, 이 문서의 종족 설명은 경영, 직원 운영, 방어, 원정에서의 기능 중심으로 정리한다.

초기 로스터는 다음 역할을 나눠 가진다.

```text
대량 방문/오염 관리: 슬라임
전투/훈련/수리: 오크
장기 체류/연구/마력/공포: 뱀파이어
빠른 보충/상점 운영/정찰: 수인
희귀 방문/고액 소비/마력/저주: 데몬
```

기존의 `언데드`는 너무 넓은 분류이므로, 실제 종족 컨셉은 `뱀파이어`로 좁힌다. 뱀파이어는 장기 체류, 어둠, 마력, 연구, 공포 역할을 명확하게 담당한다.

기존의 `고블린`은 빠른 물류/함정/장난 계열 역할은 좋지만 도난과 장난 쪽으로 성격이 과하게 쏠린다. 그 역할은 `수인`으로 옮기고, 고블린은 이후 이벤트성 방문자나 하위 변종으로 확장한다.

## 초기 종족 수

컨셉은 5종을 잡는다.

```text
슬라임
오크
뱀파이어
수인
데몬
```

다만 첫 프로토타입에서 모두 구현하지 않는다.

첫 구현 추천:

```text
슬라임
오크
뱀파이어
```

이 3종만으로도 대량 저가 고객, 전투 고객, 연구/마력 고객의 차이를 검증할 수 있다.

수인과 데몬은 이후 확장으로 둔다.

## 종족 요약

```text
슬라임: 대량 방문, 저소비, 청소/오염, 습기, 화염 기피
오크: 고소비, 전투/훈련, 경비/수리, 난동
뱀파이어: 장기 체류, 연구/마력, 어둠 선호, 공포
수인: 빠른 이동, 보충/상점 운영, 정찰, 소동
데몬: 희귀 방문, 고액 소비, 마력/저주, 강한 전투/연구
```

## 슬라임

### 핵심 판타지

많이 오고 오래 퍼지는 저가 대량 고객층이다.

던전을 활기차게 채워주지만, 방치하면 점액과 오염 문제를 만든다.

### 고객 컨셉

- 방문 빈도 높음
- 소비 금액 낮음
- 저가 음식점과 습한 휴식 시설 선호
- 고급 시설보다 접근성과 수용량을 중시
- 화염/건조/뜨거운 시설 근처 동선 기피

### 직원 컨셉

강점:

- 청소
- 오염 처리
- 좁은 구역 작업
- 낮은 유지 부담

약점:

- 공격
- 힘
- 경비
- 무거운 재고 운반

### 방어/원정 컨셉

슬라임은 직접 싸우기보다 지연과 보조에 어울린다.

방어 역할:

- 침입자 감속 보조
- 오염/점액 구역 관리
- 파손 이후 청소와 복구 보조

원정 역할:

- 낮은 위험 지역 보조 인력
- 장기 원정보다 짧은 보급/채집형 원정에 적합

### 시설 공격 컨셉 시너지

어울리는 계열:

- 냉기/속박
- 독/부식

기피/충돌:

- 화염/연소

운영 의미:

```text
슬라임 고객층을 키우면 냉기/속박, 독/부식 기반의 지연형 던전과 잘 맞는다.
하지만 화염 계열 방어 시설을 같은 동선에 두면 만족도가 떨어진다.
```

### 사고 위험

슬라임 오염:

- 주변 시설을 점액으로 오염시킨다.
- 청소 작업을 만든다.
- 방치하면 손님 만족도와 직원 효율에 악영향을 준다.

대응:

- 청소 우선순위 직원
- 휴식방(습기)
- 충분한 저가 시설 수용량
- 화염 시설과 동선 분리

### UI 태그

```text
대량 방문
저소비
습기 선호
화염 기피
청소 적성
오염 위험
```

## 오크

### 핵심 판타지

돈을 많이 쓰고, 많이 먹고, 싸움과 훈련을 좋아하는 전투 고객층이다.

잘 관리하면 강한 경비/원정 인력이 되지만, 불만이 쌓이면 시설을 부수는 위험한 내부 리스크가 된다.

### 고객 컨셉

- 소비 금액 높음
- 고기 식당, 무기점, 훈련장 선호
- 조용하고 장식 위주의 고급 시설에는 관심 낮음
- 대기 시간이 길거나 식사/훈련 욕구가 막히면 불만 증가

### 직원 컨셉

강점:

- 공격
- 힘
- 경비
- 수리
- 훈련

약점:

- 연구
- 청소
- 섬세한 근무

### 방어/원정 컨셉

방어 역할:

- 침입자와 직접 교전
- 반란 제압
- 파손 시설 수리 보조
- 병목 구역 경비

원정 역할:

- 전투 원정 핵심 인력
- 위험 지역 돌파
- 약탈/제압형 원정에 적합

### 시설 공격 컨셉 시너지

어울리는 계열:

- 물리/가시
- 화염/연소

운영 의미:

```text
오크 고객층을 키우면 물리 피해와 화염 피해를 중심으로 한 공격적인 던전이 어울린다.
훈련장, 고기 식당, 경비실이 오크 영입 가치와 방어력을 동시에 만든다.
```

### 사고 위험

오크 난동:

- 가까운 시설을 파손시킨다.
- 주변 손님을 겁먹게 하거나 동선을 막는다.
- 경비 인력이 부족하면 반란/내부 전투로 커질 수 있다.

대응:

- 고기 식당 재고 유지
- 훈련장 제공
- 경비 인력 배치
- 혼잡 완화
- 위험한 오크 직원은 핵심 시설 주변에 무리하게 몰지 않기

### UI 태그

```text
고소비
전투 선호
고기 선호
경비 적성
힘 높음
난동 위험
```

## 뱀파이어

### 핵심 판타지

식사 부담이 낮고 오래 체류하는 연구/마력 계열 고객층이다.

식사 부담은 낮지만, 어둠 시설과 전용 구역 요구가 있고 잘못 섞이면 다른 손님에게 공포를 준다.

### 고객 컨셉

- 일반 식사 욕구 낮음
- 오래 체류
- 휴식방(어둠), 어둠 시설, 마력 저장소, 연구 시설 선호
- 밝은 시설, 성스러운 장식 기피
- 일반 손님과 같은 구역에 너무 많이 섞이면 공포 문제 발생
- 혼잡하지 않은 전용 구역 선호

### 직원 컨셉

강점:

- 연구
- 마력 분석
- 장시간 근무
- 피로 저항
- 마력/연구 시설 근무

약점:

- 청소
- 운반
- 활기찬 손님 응대

### 방어/원정 컨셉

방어 역할:

- 공포/저주 계열 확장과 시너지
- 장기 경비
- 마력 시설 보조
- 침입자 약화

원정 역할:

- 장기 원정
- 마력/유물 조사
- 위험한 환경에서 안정적인 보조

### 시설 공격 컨셉 시너지

어울리는 계열:

- 독/부식
- 냉기/속박
- 공포/저주

운영 의미:

```text
뱀파이어는 독/부식, 냉기/속박 기반의 느리고 끈질긴 방어와 잘 맞는다.
공포/저주 계열이 해금되면 침입자 약화 역할이 더 강해진다.
```

### 사고 위험

뱀파이어 공포:

- 특정 손님의 만족도를 낮춘다.
- 밝은/활기찬 고객층과 충돌한다.
- 공포가 누적되면 손님 이탈이나 직원 불만으로 이어질 수 있다.

대응:

- 어둠 구역 분리
- 어둠/마력 구역을 일반 상업 구역과 나누기
- 뱀파이어 전용 동선
- 공포에 강한 고객층 중심 운영

### UI 태그

```text
식사 부담 낮음
장기 체류
연구 적성
마력 선호
어둠 선호
밝음 기피
공포 위험
```

## 수인

### 핵심 판타지

빠른 이동과 보충에 강한 물류/상점 운영 계열 고객층이다.

돈을 크게 쓰지는 않지만 자주 오고, 영입하면 보충, 상점 운영, 빠른 대응에 유용하다.

### 고객 컨셉

- 방문 빈도 높음
- 소비 금액 낮음-중간
- 잡화점, 간식점, 값싼 상점, 놀이 시설 선호
- 비싼 고급 시설 기피
- 혼잡하면 장난/소동 위험 증가

### 직원 컨셉

강점:

- 이동속도
- 재고 보충
- 상점 근무
- 손재주
- 간단한 수리

약점:

- 맷집
- 힘
- 정면 전투
- 복잡한 접객

### 방어/원정 컨셉

방어 역할:

- 빠른 경보 전달
- 빠른 수리
- 재고/장치 보충
- 반란이나 침입 상황의 민첩한 대응

원정 역할:

- 정찰
- 빠른 회수
- 물자 회수
- 위험도 낮은 자원지 탐색

### 시설 공격 컨셉 시너지

어울리는 계열:

- 물리/가시
- 번개/축전

운영 의미:

```text
수인은 직접 강하게 싸우기보다 빠르게 움직이며 시설과 장치를 보조하는 종족이다.
번개/축전이나 가시 함정처럼 위치 대응과 장치성이 중요한 방어 시설과 잘 맞는다.
```

### 사고 위험

수인 소동:

- 주변 손님 만족도 감소
- 짧은 시간 동안 시설 근무 방해
- 일부 재고 손실 가능
- 혼잡 구역에서 발생 확률 증가

대응:

- 감시 시설
- 혼잡 완화
- 값싼 상점 충분히 제공
- 놀이/휴식 시설 제공

### UI 태그

```text
자주 방문
저가 소비
빠름
보충 적성
상점 운영
정찰 적성
소동 위험
```

## 데몬

### 핵심 판타지

적게 오지만 큰돈을 쓰는 마력/저주 계열 고객층이다.

만족시키면 강력한 직원과 고수익을 얻지만, 요구 수준이 높고 불만 시 저주나 반란 위험이 크다.

### 고객 컨셉

- 방문 빈도 낮음
- 소비 금액 매우 높음
- 고급 상점, 마력 시설, 제단, 장식 시설 선호
- 저급 시설과 과도한 혼잡 기피
- 전용 동선과 고급 구역이 중요

### 직원 컨셉

강점:

- 공격
- 연구
- 마력
- 협상/위압
- 고급 시설 근무

약점:

- 청소
- 단순 운반
- 낮은 등급 시설 근무
- 혼잡 환경

### 방어/원정 컨셉

방어 역할:

- 강한 단일 전투력
- 마력 시설 보조
- 반란 제압 또는 협상/진정 보조
- 후반 공포/저주 계열과 시너지

원정 역할:

- 고위험 원정
- 희귀 보상 원정
- 보스급 전투 지원

### 시설 공격 컨셉 시너지

어울리는 계열:

- 화염/연소
- 번개/축전
- 공포/저주

운영 의미:

```text
데몬 고객층을 노리면 고급 시설과 마력 시설 중심의 비싼 던전이 된다.
화염과 번개 계열의 고밸류 방어 시설과 잘 어울리지만, 운영 실패 시 피해도 크다.
```

### 사고 위험

데몬 저주:

- 주변 시설 유지 부담 증가
- 손님 만족도 감소
- 특정 직원의 불만 증가
- 심하면 내부 위협으로 확산

대응:

- 고급 시설 유지
- 혼잡 완화
- 전용 동선
- 마력 시설 충분히 제공
- 위험 상태 알림 확인

### UI 태그

```text
고액 소비
희귀 방문
고급 선호
마력 적성
강한 전투
저주 위험
```

## 종족 간 관계

종족 관계는 복잡한 상성표보다 운영 감각으로 시작한다.

좋은 조합:

```text
슬라임 + 수인:
대량 저가 고객층, 수용량과 회전율 중심 운영

오크 + 수인:
활기찬 상업/함정 구역, 전투와 장치 시너지

뱀파이어 + 데몬:
마력/어둠/고급 시설 중심 운영

오크 + 뱀파이어:
전투와 장기 방어 중심 운영
```

충돌 조합:

```text
슬라임 + 화염 중심 오크/데몬 구역:
화염 시설 때문에 슬라임 만족도 하락

뱀파이어 + 일반 대량 고객 구역:
공포 때문에 일부 손님 만족도 하락

데몬 + 수인 대량 혼잡 구역:
데몬이 혼잡과 저급 시설을 싫어함

오크 + 조용한 고급 데몬 구역:
오크의 활기와 난동 위험이 데몬 동선과 충돌
```

## 시설 공격 컨셉 연결

종족과 공격 컨셉은 다음처럼 연결한다.

```text
슬라임: 냉기/속박, 독/부식 선호 / 화염 기피
오크: 물리/가시, 화염/연소 선호
뱀파이어: 독/부식, 냉기/속박, 공포/저주 선호
수인: 물리/가시, 번개/축전 선호
데몬: 화염/연소, 번개/축전, 공포/저주 선호
```

이 연결은 정답 조합이 아니라 방향성이다.

플레이어는 어떤 고객층을 키우는지에 따라 자연스럽게 다른 방어 시설을 고르게 된다.

## 초기 프로토타입 범위

첫 구현에서는 다음만 만든다.

- 종족 3개: 슬라임, 오크, 뱀파이어
- 종족별 선호 시설 2개
- 종족별 기피 요소 1개
- 종족별 직원 적성 2개
- 종족별 사고 1개
- 종족별 시설 공격 컨셉 연결 1-2개

초기 연결:

```text
슬라임:
- 선호 시설: 저가 음식점, 휴식방(습기)
- 기피: 화염 시설 근처 동선
- 직원 적성: 청소, 오염 처리
- 사고: 오염
- 공격 컨셉 연결: 냉기/속박

오크:
- 선호 시설: 고기 식당, 훈련장
- 기피: 조용한 고급 장식 구역
- 직원 적성: 경비, 수리
- 사고: 난동
- 공격 컨셉 연결: 물리/가시, 화염/연소

뱀파이어:
- 선호 시설: 휴식방(어둠), 마력 저장소
- 기피: 밝은 시설
- 직원 적성: 연구, 마력 분석
- 사고: 공포
- 공격 컨셉 연결: 독/부식, 냉기/속박
```

수인과 데몬은 데이터 구조만 고려하고, 첫 프로토타입에서는 실제 방문/영입 대상으로 넣지 않는다.
