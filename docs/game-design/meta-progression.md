# 계승과 메타 진행

DungeonStory는 한 번의 런에서 던전을 운영하고, 플레이어 캐릭터가 사망하면 런이 끝나는 구조다.

하지만 런이 끝나도 모든 것이 사라지지는 않는다.

플레이어는 얼마나 오래 버텼는지에 따라 계승 화폐를 얻고, 이 화폐로 다음 런의 선택지와 기본 역량을 강화한다.

정리하면 다음 구조다.

```text
런 시작
-> 던전 경영/침입 방어/합성/오펜스
-> 플레이어 캐릭터 사망
-> 런 결과 정산
-> 생존 성과에 따른 계승 화폐 획득
-> 계승 트리 강화
-> 다음 런 시작 선택지 증가
```

이 구조는 순수 로그라이크보다는 로그라이트식 메타 진행에 가깝다.

## 계승 화폐

계승 화폐는 런 종료 시 지급된다.

이 화폐는 현재 던전의 돈이나 재고가 아니라, 다음 런으로 이어지는 메타 자원이다.

획득 기준의 핵심은 `얼마나 오래 버텼는가`다.

주요 기준:

- 생존 시간
- 운영일 정산 횟수
- 막아낸 침입 수
- 도달한 최대 위협 단계
- 최종 침입 강도

보조 기준:

- 처음 발견한 시설
- 처음 해금한 조합식
- 처음 만난 몬스터/침입자 정보
- 오펜스 원정 성공
- 특수 이벤트 해결

기본 계산 방향:

```text
계승 화폐
= 생존 보상
+ 침입 방어 보상
+ 최초 발견 보너스
+ 오펜스 성과 보너스
+ 난이도 배율
```

생존 보상이 가장 큰 비중을 가져야 한다.

최초 발견이나 오펜스 보너스는 좋은 보조 보상이지만, 짧게 죽고 보너스만 챙기는 플레이가 최적이 되면 안 된다.

## 런 결과 정산

런이 끝나면 결과 화면에서 계승 화폐 획득 근거를 보여준다.

예시:

```text
생존 운영일: 18
막아낸 침입: 5회
최대 위협 단계: 4
최고 던전 가치: 12,400
발견한 시설: 6개
해금한 조합식: 3개
오펜스 성공: 2회
난이도 배율: x1.0

획득 계승 화폐: 340
```

결과 화면은 처방을 주는 화면이 아니라, 이번 런이 어떤 방식으로 성장했고 어디까지 버텼는지 정리하는 화면이다.

## 계승 강화 방향

계승 강화는 단순히 모든 수치를 영구적으로 올리는 방식이면 안 된다.

핵심은 다음 런의 선택지를 늘리고, 초반 반복 피로를 줄이고, 새로운 빌드를 시도하게 만드는 것이다.

강화 축:

```text
시작 선택지 확장
설계도/조합식 계승
사장 기본 능력 강화
상점/드래프트 보정
도감/정보망 강화
오펜스/정찰 강화
```

### 시작 선택지 확장

다음 런 시작 시 고를 수 있는 선택지가 늘어난다.

예시:

```text
시작 시설 후보 +1
시작 몬스터 손님층 후보 +1
시작 사장 종족 후보 +1
시작 사장 특성 후보 +1
시작 직원 후보 등장
시작 설계도 후보 등장
시작 방어 시설 후보 등장
```

이 강화는 직접적인 깡스탯보다 덱빌딩 재미에 더 잘 맞는다.

플레이어가 매 런 다른 방향을 시도할 수 있게 한다.

### 설계도/조합식 계승

일반 조합식은 공개되어 있고, 특수 조합식은 설계도 연구로 해금된다.

계승 강화는 이 구조와 연결된다.

예시:

```text
연구한 특수 조합식 영구 기록
특정 시설군의 1성 조합식 기본 공개
특정 시설군의 2성 조합식 일부 공개
특수 설계도 보관 슬롯 증가
다음 런 시작 시 설계도 1개 선택
```

주의:

- 모든 특수 조합식을 영구 해금하면 런마다 발견하는 재미가 줄어든다.
- 중요한 특수 조합식은 영구 기록되더라도, 실제 제작에는 런 안에서 시설과 조건을 다시 마련해야 한다.

즉 조합식 지식은 계승되지만, 시설과 재고와 인력은 다시 모아야 한다.

### 사장 기본 능력 강화

플레이어 캐릭터는 죽으면 런이 끝나는 핵심 코어다.

따라서 사장 강화는 계승 트리의 중요한 축이 될 수 있다.

예시:

```text
최대 체력 소폭 증가
최종 교전 생존력 증가
사장방 기본 방어 보정
초기 명령/지시 슬롯 증가
초기 연구 속도 보정
초기 직원 통솔 보정
시작 사장 종족 해금
시작 사장 개인 특성 후보 증가
```

다만 사장 강화는 너무 크게 주면 방어 설계의 의미가 줄어든다.

방향:

```text
초반 안정성은 조금 올린다.
최종 교전에서 한 번 더 버틸 여지를 준다.
하지만 침입을 무시할 정도의 영구 전투력은 주지 않는다.
```

### 상점/드래프트 보정

운영일 상점과 시작 선택지의 운을 완화한다.

예시:

```text
운영일 상점 상품 후보 +1
상점 새로고침 비용 감소
1성 기본 시설 구매 목록 확장
특정 시설군 등장 확률 증가
시작 드래프트 다시 뽑기 횟수 +1
```

이 강화는 플레이어가 원하는 빌드를 어느 정도 노릴 수 있게 해준다.

### 도감/정보망 강화

관찰한 정보와 도감은 계승된다.

예시:

```text
침입자 약점 기록 유지
몬스터 선호/기피 정보 유지
시설 효과 설명 유지
특수 조합식 힌트 보존
침입 전 경고 정확도 증가
```

이 축은 밸런스를 크게 망치지 않으면서도, 플레이어가 플레이할수록 똑똑해지는 감각을 준다.

### 오펜스/정찰 강화

오펜스가 해금된 뒤에는 월드맵과 정찰 정보 일부가 계승될 수 있다.

예시:

```text
시작 스캔 범위 증가
가까운 원정 대상 일부 표시
원정 위험도 예측 정확도 증가
희귀 시설 보상지 힌트 증가
원정대 편성 슬롯 증가
```

오펜스 계승은 후반 계승 트리로 둔다.

초반부터 열면 경영/디펜스 기본 루프가 흐려질 수 있다.

## 계승되지 않는 것

다음 요소는 기본적으로 계승하지 않는다.

- 현재 던전의 돈
- 현재 재고
- 배치된 시설
- 런 중 고용한 대부분의 직원
- 시설의 현재 레벨과 위치
- 현재 침입 위협도
- 현재 상점 판매 목록

이 요소들이 그대로 계승되면 런의 긴장감이 약해진다.

대신 플레이어는 다음 런에서 더 좋은 선택지를 가지고 다시 시작한다.

## 강화 밸런스 원칙

계승 강화는 있어야 하지만, 게임을 자동으로 이기게 만들면 안 된다.

원칙:

- 화폐 획득의 중심은 생존 시간이다.
- 직접 전투력 강화는 작고 단계적으로 제한한다.
- 선택지 확장과 정보 계승을 주요 보상으로 둔다.
- 초반 반복 피로를 줄이는 강화는 적극적으로 허용한다.
- 높은 난이도나 후반 위협은 메타 강화만으로 무시할 수 없어야 한다.

좋은 계승 강화:

```text
다음 런에 시작 시설 후보가 1개 더 뜬다.
특수 조합식 하나를 기억한다.
상점 후보가 1칸 늘어난다.
사장방 최종 교전에서 조금 더 버틴다.
침입자 약점 정보가 더 빨리 보인다.
```

위험한 계승 강화:

```text
모든 시설 피해량 영구 +50%
모든 손님 소비 금액 영구 +100%
사장 체력 대폭 증가
초기 자금 과도 증가
처음부터 고등급 시설 지급
```

강화는 플레이어를 강하게 만들되, 매 런의 배치와 선택을 대체하면 안 된다.

## 초기 구현 추천

첫 구현에서는 계승 시스템도 작게 시작한다.

초기 계승 요소:

- 생존 성과 기반 계승 화폐 지급
- 간단한 런 결과 화면
- 계승 트리 3갈래

초기 계승 트리:

```text
운영 지식:
상점 후보 +1, 기본 구매 목록 확장

설계 보존:
특수 조합식 기록 슬롯, 시작 설계도 후보

사장 생존:
사장 체력 소폭 증가, 사장방 최종 교전 보정
```

오펜스/정찰 계승은 오펜스 시스템이 안정된 뒤 추가한다.
